esxi-safe-shutdown

ESXi safe shutdown script

Download this project as a .zip file Download this project as a tar.gz file

What is it?

A tool for VMware ESXi servers to safely shutdown all running VMs (even ones that don't have VMware Tools installed), before powering off the ESXi host.

Where will it work?

I've tested it with ESXi 5.1, it'll probably work with 5.0 as well but I haven't tried that. I don't think it'll work on 4.x though. Let me know how you get on!

Author and Contributors

Aaron B. Russell (@arussell)

Reporting problems

Use the issue tracker. Pull requests are welcome, though please remember that ESXi doesn't run bash so ensure your code is compatible with basic sh.
